Can work with both types of import/export in HTML.

You can use export on certain function or variables which makes the functions or variables available for other parts of the code. You use the import to get access to the bring the functions or variables into another module. A module in javascript is a file containing related code.

Can import/export in both ways in Node.js

In node.js you can import/export in 2 ways. There is require which is commonJs and there is import which is type modules(ECMAscript). The commonJs modules are loaded synchronously while type modules are loaded asynchronously. XSS: Understands the problem and how to avoid it.

XSS or Cross-Site-Scripting is a security problem that we have in the browser. When we use innerHtml then any string is treated by html code and for that reason the attack can insert some dangerous code that could lead to XSS attack. That is why we use innerText instead.
