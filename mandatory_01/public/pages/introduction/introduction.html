Work with Git in the Terminal

If you want to clone a project then you have to use the command:

git clone gitlink

If you want to convert it into a repository then you need to use the command:

git init

If you want to add a remote to your repository then you can use the command:

The remote tells Git where to pull and push from, and it is tied to the specific folder on your computer.

git remote add origin path

If you want to add to the project then you need to use the command:

The -A is all files.

git add -A

Then you have to use the command.

The -m allows you to write a commit message.

git commit -m "commit message"

If you want to push then you can use the command:

git push

Datatypes in javascript:

Javascript has several data types which are primitive or non-primitive.

    Number: it is a value and it can be with commas so a number can be 3 or 3.40.
    String: Textual data where you wrap it around double or single quote, "Hello World".
    Boolean: a logical value that returns true or false.
    Null: Which returns a null value.
    Undefined: A undefined datatype is a type that is uninitialized or is undefined

Then we also have three non-primitive types:

    Object: An object is a collection of data where you wrap it around brackets {}
    Array: Array is an orded list where you can store som values in that list. You write arrays in square brackets[].
    Function: A function is reusable code that does something specific.

Explain node and how to run files

Node is an open-source back-end JavaScript runtime environment. This means that you can run JavaScript code outside of the web-browser. Node is used for server-side web-applications, which means that the code gets executed on a server instead rather than on the client.
