Understands the difference between client-side redirection and server-side.

In client side redirection the browser redirect the user to a new URL, without having any contact with the server. With that you can use "Window.location.href = "URL". In server side redirection the redirection is happening on the server side, which makes it more reliable and we can use the "res.redirect".
